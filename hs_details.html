Project information

  • Name: PureService Coop
  • Languages: HTML5 / CSS3 / Javascript / Bootstrap / PHP / MySQL / Laravel-Livewire

Description

PureServiceCoop is a system that provides users with a choice of home services. Contains 3 modules, user module, service provider and administrator. The user can research about the services provided and contact the company while the administrator manages the content, services, service categories, service providers, and contacts.
